ส่วนที่ 1 โครงสร้างไฟล์ในโฟลเดอร์โปรเจ็กต์
	- โฟลเดอร์ doc/	
		-โฟลเดอร์ PPT การนำเสนอ/ (จัดเก็บ สไลด์ในการนำเสนอ pptx และ PDF)
		- โฟลเดอร์	 รูปเล่ม/ (จัดเก็บรายงาน (docx และ PDF))	
	- โฟลเดอร์ source/ จัดเก็บไฟล์ที่เกี่ยวข้องในโครงงานทั้งหมด
		- Bangbar.zip เป็น zip ตัวโฟลเดอร์เก็บ code การทำงานทั้งหมดของตัวโปรเจ็ค
		- ถ้าแตกไฟล์ จะได้โฟลเดอร์ Bangbar ภายในโฟลเดอร์Bangbarจะมีโฟลเดอร์ uploads เพื่อเก็บรูปต่างๆ นอกจากนั้นก็จะเป็นไฟล์css และ php
						
	- โฟลเดอร์ sql/  จัดเก็บ sql scripts สำหรับสร้างฐานข้อมูล
		- signup.sql 

ส่วนที่ 2 การติดตั้งโครงงาน 
	-ขั้นที่ 1 ให้ทำการโหลดและติดตั้งโปแกรม XAMPP จาก https://www.apachefriends.org/download.html
	-ขั้นที่ 2 ให้ทำการแตกไฟล์ Bangbar.zip ที่อยู่ในโฟเดอร์ source/ และ นำโฟลเดอร์ที่ได้จากการแตกไฟล์ไปไว้ใน C:\xampp/htdocs/
	-ขั้นที่ 3 หลังจากที่ได้ติดตั้ง  XAMPP แล้วให้ทำการเปิดตัวโปแกรมขึ้นมาและ Start ตัว Apache และ MySQL 
	-ขั้นที่ 4 นำสคริปต์ signup.sql สำหรับสร้างฐานข้อมูลรันใน PHPMyAdmin เพื่อสร้างฐานข้อมูลและตารางที่เกี่ยวข้อง
	-ขั้นที่ 5 ไปยัง http://localhost/Bangbar/login.php เพื่อไปยังหน้า login ของตัวเว็บโครงงาน สามารถregisterผู้ใช้งานใหม่ หรือ 
	ใส่username: admin password:admin
